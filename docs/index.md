# OpenClinic Documentation

OpenClinic is an open-source medical records system built with Django.

## Documentation Structure

This documentation follows the [Diátaxis Framework](https://diataxis.frameworks.io/) to organize content by user need:

```
docs/
├── index.md              # Documentation home
├── tutorials/            # Learning-oriented tutorials
│   ├── getting-started.md
│   └── patient-management.md
├── how-to/              # Problem-oriented guides
│   ├── install.md
│   ├── configure.md
│   └── development.md
├── reference/           # Information-oriented reference
│   ├── architecture.md
│   ├── models.md
│   └── api.md
└── explanation/          # Understanding-oriented content
    ├── concepts.md
    └── security.md
```
